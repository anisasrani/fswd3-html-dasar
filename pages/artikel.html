PENGENALAN HTLM DAN CSS

Apa Itu HTML?

HTML merupakan bahasa yang digunakan untuk mendeskripsikan halaman web. HTML (Hyper Text Markup Language) merupakan bahasa formatting yang digunakan untuk membuat halaman website. Pada dunia pemrograman berbasis website (Web Programming), HTML menjadi pondasi dasar pada halaman website, selain CSS dan Javascript. sebuah file HTML di di simpan dengan ekstensi .html (dot html). dan dapat di eksekusi atau diakses menggunakan web browser, seperti Google Chrome dan Mozilla Firefox.

Syarat Penulisan HTML

(1) Penulisan tag diapit tanda kurung siku “<”. (2) Tag HTML secara normal berpasangan (3) Tag HTML tidak case sensitive.

Markup atau Tanda

Secara garis besar, terdapat 4 jenis elemen dari HTML: (1) Structural. Tanda yang menentukan level atau tingkatan dari sebuah teks (2) Presentational. Tanda yang menentukan tampilan dari sebuah teks tidak peduli dengan level dari teks tersebut (3) Hypertext. Tanda yang menunjukkan pranala ke bagian dari dokumen tersebut atau pranala ke dokumen lain (4) Elemen widget yang membuat objek-objek lain seperti tombol, list, dan garis horizontal

Elemen < html >

Hierarki elemen yang paling atas pada berkas HTML adalah elemen HTML. Elemen ini memberitahu kepada browser bahwa ini sebuah berkas HTML, serta menjadi root dari sebuah berkasnya itu. Seluruh elemen lainnya akan dituliskan didalam elemen ini.

Elemen < head >

Elemen < head > berfungsi sebagai tempat disimpannya informasi dari dokumen HTML.
Elemen < head > biasanya berisi elemen lain, seperti: < title >, < style >, < base >, < link >, < meta >, < script >, < noscript >

Elemen < body >

Seluruh konten pada elemen < body > akan ditampilkan pada halaman website, yaitu pada browser. Sebagai contoh, coba ketikkan kode berikut, simpan dalam format.
